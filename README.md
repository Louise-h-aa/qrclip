## Installation

```console
# clone the repo
$ git clone https://github.com/Louise-h-aa/qrclip.git

# change the working directory to qrclip
$ cd qrclip

# install the requirements
$ python3 -m pip install -r requirements.txt
```

## Usage

```console
python3 qrclip.py
```
